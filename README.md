# 🌌 LighthouseAI

### Illuminate Your Path To Mastery
URL - https://project-lighthouseai.vercel.app/
## 🚀 Overview

LighthouseAI is an AI-powered academic performance prediction platform
that helps students understand where they are headed before exams and
how to improve it.

## 🎯 Problem

Students realize performance too late. No system predicts outcomes or
provides actionable insights.

## 💡 Solution

Predicts exam percentage and gives personalized improvement strategies.

## 🧠 Features

-   Score Prediction with What-If Simulator\
-   YouTube Recommendations\
-   Notes Summarizer\
-   Study Roadmap\
-   AI Study Coach\
-   Timetable Generator

## 🧪 ML Model

-   Random Forest\
-   206 student dataset\
-   MAE: ±4.6%\
-   R²: 0.569

## 🛠 Tech Stack

React, Node.js, Flask, Gemini API, Vercel, Render
